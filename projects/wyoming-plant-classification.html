Wyoming Plant Classification System

Built an end-to-end pipeline for multi-class invasive plant detection using GroundedSAM for zero-shot annotation and YOLOv8 for object detection, achieving 85% mAP@50 across 3 plant species. Automated dataset generation, multi-class merging, and video inference with OpenCV.
