NYC Citi Bike Urban Mobility Analysis

Large-scale analysis of 70M+ trip records to uncover urban mobility patterns, commuter segments, and station utilization trends.

Tech Stack

  • Apache Spark
  • FP-Growth
  • Power BI
  • Python
  • Pandas

Project Overview

Processed 70M+ trip records using Apache Spark to analyze large-scale user behavior and temporal demand patterns. Applied FP-Growth mining to identify recurring usage segments (commuter vs. off-peak users) and developed Power BI dashboards to visualize spatio-temporal trends and station utilization.
